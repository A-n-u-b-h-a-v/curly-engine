# Mojito - React Animated Landing Page

Mojito is a visually engaging and highly interactive landing page built with modern web technologies. It features smooth scrolling, dynamic scroll-triggered animations, and a responsive design, making it perfect for showcasing creative content or hospitality brands.

## üöÄ Tech Stack

This project uses a powerful combination of tools to deliver a premium user experience:

- **[React](https://react.dev/)**: The library for web and native user interfaces.
- **[Vite](https://vitejs.dev/)**: A fast frontend tooling build tool.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.
- **[GSAP (GreenSock Animation Platform)](https://gsap.com/)**: The standard for modern web animation, used here with `ScrollTrigger` for scroll-based effects.
- **[Lenis](https://github.com/studio-freight/lenis)**: A lightweight library for smooth momentum scrolling.

## ‚ú® Features

- **Smooth Scrolling**: Integrated `Lenis` for a fluid, high-end scrolling feel.
- **Scroll Animations**: Dynamic elements that reveal, move, or transform as you scroll, powered by `GSAP ScrollTrigger`.
- **Responsive Layout**: Adapts seamlessly to diverse screen sizes.
- **Modern UI**: Clean and aesthetic design components including:
  - **Hero Section**: Impactful first impression.
  - **Art & About**: Creative showcase sections.
  - **Cocktail & Menu**: Highlighted offerings.
  - **Contact**: Easy way to get in touch.
