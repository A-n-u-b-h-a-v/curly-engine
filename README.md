## üõ†Ô∏è Comprehensive Tech Stack

This portfolio demonstrates proficiency across the entire web development spectrum.

### Frontend Ecosystem
- **Core Frameworks**: [React](https://react.dev/), [Next.js](https://nextjs.org/) (App Router & Pages Router).
- **Languages**: TypeScript (extensive usage for type safety), JavaScript (ES6+).
- **Build Tools**: [Vite](https://vitejs.dev/) (for blazing fast dev servers), Webpack.
- **Styling**:
  - **Tailwind CSS**: Used in almost every project for utility-first styling.
  - **GSAP**: Industry-standard library for complex, high-performance animations.
  - **Three.js / React Three Fiber**: For immersive 3D web experiences.
  - **Framer Motion**: For declarative component animations.
  - **Glassmorphism/CSS3**: Advanced CSS techniques.

### Backend & Database
- **Runtime**: [Node.js](https://nodejs.org/), [Bun](https://bun.sh/) (Serverless/Edge compatible).
- **Frameworks**:
  - **Express.js**: Standard REST API servers.
  - **Hono**: High-performance web framework for Edge environments.
  - **Next.js API Routes**: Serverless functions.
- **Databases**:
  - **MongoDB**: NoSQL database for flexible data structures.
  - **PostgreSQL**: Relational database for structured data.
- **ORMs**:
  - **Prisma**: Next-generation Node.js and TypeScript ORM.
  - **Mongoose**: Elegant MongoDB object modeling.

### Real-Time & Networking
- **Socket.io**: For bidirectional event-based communication (Chat, Tracker, Chess).
- **WebSockets (Native)**: Low-level real-time protocols.
- **Stream SDK**: For enterprise-grade video and audio streaming.

### Tools & DevOps
- **Authentication**: [Clerk](https://clerk.com/) (User Management), JWT (access/refresh tokens), NextAuth.
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/), React Context API, Zustand (implied).
- **Validation**: [Zod](https://zod.dev/) (Schema validation).
- **Deployment**: Vercel, Cloudflare Workers.
