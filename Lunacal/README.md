# Lunacal - Glassmorphism UI Demo

Lunacal is a sleek, modern UI demonstration built with React and Tailwind CSS. It features a sophisticated dark mode aesthetic designed with glassmorphism effects, gradient backgrounds, and interactive components like tab panels and sliders.

## üöÄ Tech Stack

- **[React](https://react.dev/)**: For component-based UI architecture.
- **[Tailwind CSS](https://tailwindcss.com/)**: For advanced styling including gradients, shadows, and arbitrary values.
- **[Framer Motion](https://www.framer.com/motion/)**: For smooth component animations.
- **[Lucide React](https://lucide.dev/)**: For clean, consistent icons.
- **[Vite](https://vitejs.dev/)**: For a fast development experience.

## ‚ú® Features

- **Glassmorphism Design**: Extensive use of backdrop filters, semi-transparent backgrounds, and subtle borders to create a premium glass-like effect.
- **Interactive Components**:
  - **TabPanel**: A functional tabbed interface.
  - **Slider**: A custom slider component.
- **Responsive Layout**: Flexbox-based layout that adapts to the container.
- **Modern Styling**: Complex linear gradients and box shadows for depth and texture.
